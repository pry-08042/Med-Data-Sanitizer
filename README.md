# 🛡️ Med-Data-Sanitizer

本工具包提供了一套符合深度学习规范的医学图像隐私脱敏流水线，专为研究人员在开源数据集或协作训练前进行数据合规性预处理。

## 🌟 核心特性
- PyTorch 原生支持: 采用面向对象设计，可直接作为 `torch.utils.data.Dataset` 的 transform 模块。
- 视觉脱敏: 自动识别并清除 ISIC 2018 等数据集中的人工标记与边缘噪声。
- 差分隐私 (DP): 实现了可调控隐私预算（Epsilon）的拉普拉斯加噪机制。

## 📈 持续更新路线图 (Roadmap)

本项目将持续进行以下技术迭代：
- [ ] v1.1 : 增加对 DICOM 医疗格式元数据的深度擦除功能。
- [ ] v1.2 : 引入隐私评估模块，对比不同隐私预算对 U-Net/Mamba 模型 Dice 系数的影响。
- [ ] v2.0 : 开发针对成员推理攻击（MIA）的防御有效性量化测试工具。
